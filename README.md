# Project Title

Mini Todo App

## Description

This is the mini simple Todo App by Nhat Nam

## Getting Started

### Dependencies

* Node 12.x or higher.
* Yarn 1.22.0.

### Installing dependencies

* Yarn is recommended to install the dependencies.
```
yarn install
```

### Start the project
* Run this command below
```
yarn start
```
The website is served at port 8080.

### Build the project

* Run this command below
```
yarn build
```
